Om Tesla Handle

Förbättra din Tesla Model 3 eller Y med vårt svenskt designade handtag tillverkat av miljövänlig PLA. Detta handtag ger extra komfort för dagliga eller långa resor. När du har använt det kommer du fråga dig själv, "Varför skaffade jag inte detta tidigare?" PLA, som kommer från förnybara resurser som majsstärkelse, bryts ner på 3 till 6 månader och minskar miljöpåverkan avsevärt jämfört med traditionella plaster som tar tusentals år att brytas ner. Komplettera ditt älskade fordon med vårt högkvalitativa, biologiskt nedbrytbara handtag, vilket gör det till en omtänksam gåva för alla Tesla-ägare.

Vi har testat vårt PLA-material vid temperaturer upp till +50°C. Högre temperaturer kan dock orsaka deformation. Om du förväntar dig att din bil kommer att utsättas för temperaturer över +60°C under längre perioder, kan du överväga att välja PET-plast för dess större hållbarhet och högre värmebeständighet. Tänk på att medan PET erbjuder förbättrad prestanda, bryts det inte ner lika lätt i miljön som PLA.

Hur man installerar och tar bort

Du behöver inte göra några hål eller skada din älskade bil. Detta handtag med sugkopp lämnar inga märken när det tas bort. Vänligen följ dessa instruktioner för optimal installation:

  • Förbered ytan: Använd en våt trasa eller liknande för att ta bort olja eller damm från biltaket så att sugkoppen får bästa möjliga grepp. Var noga med att inte smutsa ner sugkoppen, eftersom det påverkar fästningen.
  • Utrymme för grepp: Lämna cirka en centimeter utrymme runt sugkoppen för optimalt grepp. Placera handtaget ungefär tre centimeter från solskyddet så att solskyddet kan öppnas och stängas fritt efter installationen.
  • Korrekt installation: Vid installation, tryck på den runda vita punkten på handtagets övre del. Eftersom takglaset är limmat på bilen, applicera nedåtgående tryck från utsidan av taket för att motverka det uppåtgående trycket på sugkoppen. Denna metod säkerställer att det inte blir någon belastning på taket och förhindrar skador på takglasets lim och tätning.
  • Ta bort handtaget: Använd ett plastverktyg för att trycka bestämt på någon sida av sugkoppen när du tar bort handtaget. Ett kreditkort fungerar bra. Använd aldrig metall eller något hårdare än plast för att undvika permanent skada på taket. Vi ansvarar inte för eventuella skador.
  • Kontroll efter installation: Efter installationen, kontrollera att handtaget sitter fast ordentligt på taket. Det bör kunna stödja vikten av din hand. Kontrollera att solskyddet fortfarande kan öppnas och stängas fritt.
